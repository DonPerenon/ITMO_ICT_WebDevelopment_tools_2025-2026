# Лабораторная работа №1 — Обзор

**Дисциплина:** Инструментальные средства веб-разработки  
**Тема:** Реализация серверного приложения FastAPI  
**Вариант:** Сервис личных финансов

## Реализованные практики

| Практика | Описание |
|----------|----------|
| [1.1](practice1.md) | Базовое FastAPI-приложение, in-memory БД, CRUD, Pydantic-модели |
