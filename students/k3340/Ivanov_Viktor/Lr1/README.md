# ЛР1 — FastAPI (вариант: сервис личных финансов)

В каталоге три практики, оформленные отдельными папками:

- `practice1` — Практика 1.1 (базовый FastAPI + временная БД + Pydantic)
- `practice2` — Практика 1.2 (PostgreSQL + SQLModel + CRUD + связи)
- `practice3` — Практика 1.3 (структура проекта + Alembic + .env + JWT-аутентификация)

Каждая папка содержит свой `README.md` с командами запуска.

## Отчет (GitHub Pages)

- Страница отчета: `docs/index.md`
- Базовая конфигурация GitHub Pages: `docs/_config.yml`
